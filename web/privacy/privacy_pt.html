Política de privacidade
1. Como condição para usar os Serviços é fazer um registro, durante o qual o Cliente deve fornecer um endereço de e-mail. O fornecimento de um endereço de e-mail não serve para estabelecer a identidade do Cliente, mas apenas para criar uma conta de usuário e oferecer a possibilidade de restaurar sua funcionalidade no caso de, por exemplo, perda desenha. Caso o endereço de e-mail fornecido pelo Cliente contenha dados pessoais, tais dados do Usuário serão processados ​​de acordo com os termos estabelecidos na Parte V deste Regulamento. 2. O administrador dos dados pessoais dos destinatários é o provedor de serviços, ou seja, a AC SOFTWARE Ltda com sede em Paczków (48-370), ul. Armia Krajowa 2, registrado pelo Tribunal Distrital de Opole, 8ª Divisão Comercial do Registo do Tribunal Nacional, No. KRS0000314809, NIP 7532365220, REGON 160217758. 3. O administrador processa os dados pessoais de acordo com as leis e regulamentos aplicáveis, e mais especificamente, com o uso de meios organizacionais e técnicos apropriados para proteger tais dados. 4. Os dados pessoais são processados ​​com base e a fim de concluir e implementar o contrato, ou seja, para a prestação dos serviços indicados na Parte II deste Regulamento. 5. Os dados pessoais também podem ser processados ​​com base e com a finalidade de: 5.1. satisfazer os interesses justificados realizados pelo Administrador, incluindo a comercialização direta de serviços próprios, 5.2. com base em um determinado consentimento - para o propósito indicado no conteúdo do consentimento concedido. 6. O fornecimento de dados pessoais é voluntário, no entanto, fornecer um endereço de e-mail é necessário para a conclusão e execução do contrato. 7. Os dados pessoais não serão compartilhados com ninguém. Os dados pessoais são confiados por contrato para processamento a entidades cujos serviços são utilizados pelo Administrador no âmbito do seu processamento (em conformidade com o artigo 28 do Regulamento Geral sobre a Proteção de Dados Pessoais de 27/04/2016 - RODO), ou seja, processadores, em particular prestadores de serviços de TI - na medida do necessário para o desempenho adequado de tais serviços. O administrador de serviços de TI selecionado pelo fornecedor garante a segurança apropriada de dados pessoais, incluindo o armazenamento de dados em servidores localizados no território de Estados-Membros da União Europeia e/ou os reconhecidos pela Comissão Europeia como tendo um nível suficiente de proteção de dados pessoais - esta é a única maneira que dados pessoais podem ser transferidos para um país terceiro. 8. Os dados pessoais serão armazenados durante o período de vigência do contrato e após a sua conclusão - pelo período necessário para estabelecer, investigar ou defender pedidos - de acordo com a lei vigente; no caso de processamento de dados com base no consentimento concedido - até a retirada deste consentimento. 9. Os dados pessoais não serão processados ​​pelo Administrador na forma de criação de perfil, decisões não serão tomadas de maneira automatizada. 10. O cliente tem o direito de: pedido do acesso de Administrador aos seus dados pessoais e direito de correção, deleção (nos casos referidos no artigo 17 do RGPD), limitações de processamento (nos casos referidos no artigo 18 do RGPD), direito de transferir os dados (nos casos referidos no artigo 20 do RGPD) e - no caso de consentimento para o processamento de dados - o direito de retirar o seu consentimento em qualquer momento, sem afetar a legalidade do tratamento efetuado com base na mesma antes da retirada. Correções de serviço podem ser feitas enviando informações relevantes para o endereço de e-mail do administrador: office@acsoftware.pl 11. O Destinatário tem o direito de apresentar uma queixa ao Presidente do Gabinete de Proteção de Dados Pessoais, caso considere que o tratamento de dados pessoais relativos a ele/ela viola as disposições do RGPD 12. Informações sobre o direito de objeção o Destinatário do Serviço tem o direito de se opor ao processamento de dados pessoais: para (e no caso de) processamento de dados pessoais com base no cumprimento de interesses legítimos realizados pelo Administrador - por motivos relacionados à sua situação específica; para (e no caso de) processamento de dados para marketing direto - a qualquer momento. O Destinatário do Serviço pode apresentar uma objeção enviando informações apropriadas para o endereço de e-mail do administrador: office@acsoftware.pl
